# Pokedex App

A simple Android app that fetches and displays Pokémon data in real time from the public [PokeAPI](https://pokeapi.co).

## Technologies
- Kotlin
- REST API

## Features
- View list of Pokémon
- Search by name
- View detailed stats and images
